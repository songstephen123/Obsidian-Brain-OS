# 07-WORK-CONTEXT

> Work Context — Your Professional Experience Accumulator

---

## English

This directory is your **work context accumulation zone**. It helps AI better understand your specific working environment, professional relationships, and past experiences — so AI can give more relevant, personalized advice when handling work-related tasks.

### Relationship with 03-KNOWLEDGE

| Directory | Content Type | AI Access |
|-----------|-------------|-----------|
| `03-KNOWLEDGE/` | Domain knowledge, principles, patterns | AI can reference freely |
| `07-WORK-CONTEXT/` | Your specific experience, decisions, relationships | AI references when handling work topics |

Think of `03-KNOWLEDGE/` as what you know about the world; `07-WORK-CONTEXT/` as what you've done in your world.

### Directory Structure

```
07-WORK-CONTEXT/
├── 01-CLIENTS/             # Client information, relationship notes, account history
├── 02-DECISIONS/          # Major decision records with context and rationale
├── 03-SUCCESS-PATTERNS/   # Patterns of what works well for you
├── 04-FAILURE-LESSONS/    # Lessons learned from setbacks and mistakes
└── 05-STAKEHOLDERS/       # Key stakeholder reminders and relationship notes
```

### Usage Guidelines

- AI may reference this directory when you're working on related topics
- Keep entries factual and concise — AI needs clear signal, not long narratives
- Update after significant work events: new decisions, lessons learned, stakeholder changes

## 中文

这个目录是你的**工作上下文积累区**，帮助 AI 更好地理解你的具体工作环境、职业关系和过往经验——从而在工作相关任务中给出更相关、更个性化的建议。

### 与 03-KNOWLEDGE 的关系

| 目录 | 内容类型 | AI 访问方式 |
|------|---------|-----------|
| `03-KNOWLEDGE/` | 领域知识、原理、模式 | AI 可自由引用 |
| `07-WORK-CONTEXT/` | 你的具体经验、决策、关系 | AI 处理工作时引用 |

可以把 `03-KNOWLEDGE/` 理解为"你知道的世界规律"，`07-WORK-CONTEXT/` 理解为"你在你的世界做过的事"。

### 目录结构

```
07-WORK-CONTEXT/
├── 01-CLIENTS/             # 客户信息、关系备忘、账户历史
├── 02-DECISIONS/           # 重大决策记录（含背景和原因）
├── 03-SUCCESS-PATTERNS/    # 你的成功模式
├── 04-FAILURE-LESSONS/     # 失败教训和反思
└── 05-STAKEHOLDERS/        # 关键干系人备忘
```

### 使用建议

- AI 在处理相关工作时可能会引用此目录
- 条目保持事实性和简洁性——AI 需要清晰的信号，而非长篇叙事
- 在重要工作事件后更新：新决策、经验教训、干系人变化
