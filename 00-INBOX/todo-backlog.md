# 待办清单

> 这是所有待办事项的唯一真相源。AI 会在每天 07:00 的驾驶舱中读取并处理这个文件。

## 使用说明

添加新事项时使用以下格式：

```markdown
- [ ] [P1] 事项标题
  - 领域：xxx
  - 重要性：高/中/低
  - 紧急性：高/中/低
  - 下一动作：具体动作
  - 截止：YYYY-MM-DD
```

## 优先级说明

| 级别 | 含义 |
|------|------|
| P0 | 今天必须完成，出现在今日 Top 3 |
| P1 | 本周推进，排入周计划 |
| P2 | 本月安排，加入月里程碑 |
| P3 | 需要判断，留在待办池直到评估 |
